## Features

**Scalable Architecture**

We have designed the system with microservices architecture, which allows us to scale different components of the system independently based on the demand. This allows us to handle a large number of users and requests without any performance issues.

**Real-time Matching**
We have implemented a real-time matching algorithm with strategy pattern that matches clients with drivers based on their location, budget, and preferences. This allows us to provide a fast and efficient service for our users.

**Low Maintenance**
We have designed our platform to be easy to maintain and update. We use modern development practices and tools to ensure that our code is well-structured and maintainable.
