# **SwiftMove Idea Overview**

## **Introduction**

**Name:** SwiftMove

A platform built *by students, for students* that focuses on fairness, control, and affordability in local transportation.

---

## **Core Concept: Fair Market**

### **Fair Pricing**

- Drivers set their own prices
- Ensures fair rates for both drivers and clients

### **User Control**

- Drivers have control over how much they charge
- Clients still receive competitive, transparent pricing

---

## **Key Features**

### **🎓 Student-Focused**

- Tailored to student needs and budgets

### **⚡ Convenience**

- Easy to use

### **🧩 Usability**

- Simple and intuitive interface
- Designed for quick interactions

## Unique value proposition

### **💸 Low-Cost Structure**

- Only **2% fee** to cover server costs
- No extra service fees
- Transparent pricing

### **🌍 Expansion**

- Open-source model
- Can be adapted to other cities

### **📍 Location Focus**

- Initially available in:
  - Waterloo
  - Nearby cities
